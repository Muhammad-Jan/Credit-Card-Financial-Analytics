# 💳 Credit Card Financial Analytics | End-to-End Data Analytics Project

## 📌 Project Overview

This project is a complete **End-to-End Financial Analytics Solution** built using **Python, MySQL, and Power BI**.

The objective of this project is to analyze customer spending behavior, transaction trends, revenue drivers, and customer demographics to help businesses make better financial decisions.

Instead of directly creating dashboards, the project follows a complete real-world analytics workflow starting from raw messy data to business insights.

---

# 🎯 Business Problem

Financial institutions generate millions of transactions every year.

However, raw transactional data alone cannot answer important business questions such as:

- Which customer segment generates the highest revenue?
- Which spending categories contribute the most?
- Which regions perform better?
- Which customers should be retained?
- How can the company increase profitability?

This project solves these problems through complete data analytics.

# 🔍 Exploratory Data Analysis (EDA)

EDA includes:

- Missing Value Analysis
- Duplicate Detection
- Distribution Analysis
- Outlier Detection
- Customer Demographics
- Revenue Distribution
- Spending Categories
- Customer Behavior
- Correlation Analysis

These analyses helped generate meaningful business insights before visualization.

---

# 🧹 Data Cleaning & Preprocessing

The raw datasets contained several data quality issues.

Using Python, the following preprocessing steps were performed:

- Removed extra spaces
- Standardized categorical values
- Fixed inconsistent text values
- Corrected spelling mistakes
- Handled missing values
- Removed invalid records
- Converted negative values where required
- Exported cleaned datasets for further analysis

# 📈 Key Business Insights

- Businessman customers generate the highest revenue.
- Bills, Entertainment, and Fuel are the top spending categories.
- Swipe is the most frequently used payment method.
- Customers aged between 40–50 years contribute the highest revenue.
- Texas, New York, and California are the top-performing states.
- Premium customers tend to have higher spending behavior.

# 💡 Business Recommendations

- Retain high-value customers using loyalty programs.
- Increase digital payment adoption through cashback offers.
- Launch personalized marketing campaigns.
- Improve customer engagement using category-based rewards.
- Strengthen customer retention strategies.
- Focus marketing efforts on high-performing regions.
